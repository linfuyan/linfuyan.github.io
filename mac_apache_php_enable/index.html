Mac 系统本身就预装了 Apache 和 PHP 。分别通过 apachectl -version 和 php -v 进行查看。

Apache 相关命令

启动：sudo apachectl start

停止：sudo apachectl stop

重启：sudo apachectl restart

查看版本号：sudo apachectl -v
